Time Stamp API

Node jsExpress jsherokuhtmlcss

You can get the unix time stamp of any valid date by using this api

examples of valid date strings are

  • 12 october 2015
  • just the year eg: 2015 (api will set to default on date and month )
  • just the month number eg: 11 (api will set to default on date and year )

Examples on using the api

  • https://timestamp1.herokuapp.com/12 december 2015
  • https://timestamp1.herokuapp.com/2016
  • https://timestamp1.herokuapp.com/11
